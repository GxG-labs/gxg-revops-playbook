# Positioning Sub-Skill

Use this before rewriting the profile. A high-performing LinkedIn profile starts with a target market decision.

## Choose the Primary Audience

Pick one:

| Mode | Primary Reader | Desired Action |
|------|----------------|----------------|
| Career | Recruiter, sourcer, hiring manager, referral source | InMail, interview, referral |
| B2B Sales | Buyer, founder, executive, operator, partner | Reply, call, intro, follow |
| Hybrid | Recruiter plus buyer | Clear enough for both without looking unfocused |

## Career Positioning

Create a role-search map:

| Field | Examples |
|-------|----------|
| Target titles | RevOps Manager, Sales Ops Manager, GTM Systems Lead |
| Seniority | Lead, Senior, Head of, Director, Fractional |
| Domain | B2B SaaS, PLG, enterprise, AI, fintech |
| Tools | Salesforce, HubSpot, Marketo, Apollo, Looker, SQL |
| Methods | Forecasting, attribution, lead scoring, routing, territory design |
| Proof | Metrics, team scope, systems built, companies served |

Positioning formula:

```text
{Target role} specializing in {domain/problem} using {tools/methods}, with proof in {outcome/scope}.
```

## B2B Sales Positioning

Create a buyer map:

| Field | Prompt |
|-------|--------|
| ICP | Who buys or refers this work? |
| Pain | What painful problem do they already recognize? |
| Trigger | What makes the problem urgent now? |
| Offer | What change does the user help create? |
| Proof | What evidence shows credibility? |
| CTA | What should the buyer do after reading? |

Positioning formula:

```text
I help {ICP} solve {pain} by {mechanism}, so they can {business outcome}.
```

Examples:

```text
I help B2B SaaS founders turn founder-led sales into a measurable GTM system using HubSpot, lifecycle design, and pipeline governance.
```

```text
I help CROs clean up pipeline visibility and forecast confidence before scaling headcount.
```

## Hybrid Positioning

Use when the user wants career visibility and clients.

Rules:
- Lead with category and domain, not "job seeker" or "consultant".
- Make proof broad enough to satisfy recruiters and buyers.
- Use Featured to separate career proof and client proof.
- Keep About clear about both: "open to roles and advisory projects" only if true.

Hybrid formula:

```text
{Role/category} for {domain} teams | {tools/methods} | {outcome proof}
```

## Positioning Anti-Patterns

- "Helping businesses grow" - too broad
- "Open to any opportunity" - weak search and weak trust
- "Founder / Consultant / Advisor" - no domain or outcome
- "AI enthusiast" - no buyer/job relevance
- "Strategic leader" - generic without proof
- Trying to target five unrelated roles at once
